# Kairos Advisory Website (Revamp)

A complete redesign of the Kairos Advisory Group website built with Astro and Tailwind CSS.

## Tech Stack

- **Framework:** [Astro](https://astro.build) v5.x
- **Styling:** [Tailwind CSS](https://tailwindcss.com) v4.x
- **Fonts:** Inter (sans), Playfair Display (serif)

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
